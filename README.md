

## Requirements
- Python 3.12+
- node 21.6.1

## Install 

```cmd
git clone https://github.com/Hazz-i/the-boyss
```

## Install Frontend Dependencies

```bash
cd frontend
yarn
```

## Install Backend Dependencies

```bash
cd flask
pip install flask
pip install flask-migrate
pip install flask-sqlalchemy
pip install python-dotenv
pip install pymysql
```

## Run Frontend and FLask

```bash
yarn dev
flask run
```
